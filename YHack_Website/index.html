Project Artemis

The latest in effortless trip planning

YHack 2019 · New Haven, CT · October 25-27, 2019

As a group of first-years at Yale, we came up with this project after our own inconvenient experiences exploring new areas. We realized that there was no efficient way to find the best places to explore given our time constraints. So we decided to come up with a solution— Project Artemis!

Project Artemis helps you find the best places to visit and tells you the way to visit them. We use genetic alrorithms to determine the optimal stops to turn any day trip into a hand-tailored tour.

Project Walkthrough

The FlowChart below summarizes the main algorithms and languages used in developing Project Horus.

Step 1: The user tells us where they would like to explore and how much time they have to do it.

Step 2: The Google Search API allows us to extract tourist destinations from Trip Advisor.

Step 3: We use the Google Maps API to filter the data based on location type. We also augment the data with latitude, longitude, photos, and ratings.

Step 4: We create an ajacency matrix with all of our potential locations. We model the problem as a Travelling Salesmen Problem and solve using the mlrose Python library.

Step 5: We output the optimal path to the user given the time constrainst.
